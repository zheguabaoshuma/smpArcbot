# smpArcbot plugin

## A simple chatbot to query Arcaea song scores
Running on nonebot.  
如果你是一个只想查分的玩家，首先确保安装了python，并且有如下模块：  
`imgkit` 版本`1.2.2`  
`nonebot2` 版本`2.0.0rc2`  
`nonebot-adpter-cqhttp` 版本`2.0.0b1`  
`nonebot-adapter-onebot` 版本`2.1.5`
